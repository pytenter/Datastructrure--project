# 展示视频脚本建议（5-8分钟）

1. 问题背景（30s）
- 说明新能源物流调度的现实约束：电量、载重、充电拥堵、动态任务。

2. 系统架构（60s）
- 图结构道路建模。
- 车辆/任务/充电站对象。
- 动态仿真流程。

3. 算法策略（90s）
- 最近任务优先。
- 最大任务优先。
- 协同配送开关。

4. 实验演示（120s）
- 运行命令：`python main.py`。
- 展示 small/medium/large 三种规模结果表。
- 展示 `results/summary.json` 输出。

5. 结果分析（90s）
- 对比两种策略在不同规模下的完成率、超时率、总分差异。
- 解释原因：任务权重优先与距离优先的取舍。

6. 扩展方向（60s）
- 引入全局最优解器对比。
- 引入 RL / 元启发式。
- 加入地图可视化界面。
